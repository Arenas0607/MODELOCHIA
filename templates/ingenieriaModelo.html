Ingeniería de Modelo

iniciamos con la selección del algoritmo, optando por Random Forest debido a su versatilidad. El modelo se entrena utilizando datos recolectados manualmente y enriquecidos con información de IA. Se realiza un ajuste de hiperparámetros, considerando la regresión lineal para optimizar el rendimiento. La validación se efectúa a través de una aplicación web Flask para probar su precisión. Finalmente,se lleva a cabo una evaluación exhaustiva, analizando posibles sesgos y buscando la validación de expertos en tráfico.

¿Qué buscamos lograr? Explicar cómo la predicción del tráfico puede ayudar a las empresas (puntualidad, calidad de vida de empleados) y potencialmente para marketing. Generar un modelo de machine learning usando el modelo Random Forest para que pueda analizar diferentes variables del tráfico.
