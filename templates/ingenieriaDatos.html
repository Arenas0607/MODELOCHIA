INGENIERIA DE DATOS

La Ingeniería de Datos para este proyecto se centra en seleccionar datos históricos de tráfico y variables externas relevantes, limpiar la información de errores e inconsistencias, transformar los datos normalizando valores y codificando el tiempo, y generar nuevas características útiles para la predicción. Todo este proceso se documenta para asegurar la comprensión y trazabilidad de la preparación de los datos para el modelo de Machine Learning
